## 🎯 Projenin Amacı

E-Spor Takip Merkezi, popüler e-spor oyunlarındaki (League of Legends, Counter-Strike 2, VALORANT, Dota 2) profesyonel maçları takip etmek için geliştirilmiş bir **Android uygulamasıdır**. Uygulama, bir WebView içinde çalışan tam özellikli bir Progressive Web App (PWA) barındırır.

### Temel Özellikler

- 🔴 Canlı maç takibi ve skorlar
- 📅 Bugünün maç programı
- 🔮 Yaklaşan maçlar listesi
- 🏆 Geçmiş sonuçlar
- ⭐ Favori takım yönetimi ve filtreleme
- 🔔 Akıllı bildirim sistemi (maç hatırlatma, sonuç bildirimi, günlük özet)
- 🎮 Oyun bazlı filtreleme (LoL, CS2, VALORANT, Dota 2)
- 📺 Twitch/YouTube canlı yayın bağlantıları
- 🔑 PandaScore API ile gerçek veri desteği
- 🌙 Modern dark mode tasarım

## 🛠️ Kullanılan Teknolojiler

### Android Tarafı
- **Java** — Ana programlama dili
- **Android SDK 34** — Hedef API seviyesi (Min SDK 24)
- **WebView (AndroidX WebKit)** — Web uygulamasını native sarmalayıcı içinde çalıştırma
- **AndroidX AppCompat** — Geriye dönük uyumluluk
- **ConstraintLayout** — UI düzeni
- **JavascriptInterface** — JS ↔ Android haberleşme köprüsü

### Web Tarafı (PWA)
- **HTML5** — Semantik sayfa yapısı
- **CSS3** — Responsive tasarım, dark mode, glassmorphism efektleri
- **JavaScript (ES6+)** — Uygulama mantığı, API haberleşmesi
- **PandaScore REST API** — E-spor maç verileri
- **Web Notifications API** — Bildirim sistemi
- **localStorage** — Yerel veri depolama
- **Google Fonts** — Inter, JetBrains Mono

## ⚙️ Kurulum ve Çalıştırma

### Gereksinimler
- Android Studio (Arctic Fox+)
- JDK 8+
- Android SDK 34

### Kurulum
```bash
git clone https://github.com/SalihCan14/EsportsTracker.git
```

1. Android Studio'da `esports-tracker-android` klasörünü açın
2. `esports-tracker/` içeriğini `app/src/main/assets/` altına kopyalayın
3. Gradle sync tamamlandıktan sonra **Run** ile çalıştırın

### Hazır APK
Proje dizinindeki `esports-tracker.apk` dosyası doğrudan kurulabilir.
